# MiniDevopsApp0 

A lightweight DevOps demo application built with Python, Docker, and Docker Compose.  
This project demonstrates how to containerize and run a simple Python application with basic database interaction.

---

## Features
- Python backend application
- Dockerized environment
- Docker Compose setup
- Simple database integration
- Modular project structure

---

##  Tech Stack
- Python
- Docker
- Docker Compose
- SQLite / Database module
